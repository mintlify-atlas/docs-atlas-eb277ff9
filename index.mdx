## Quick Start

<Steps>
  <Step title="Install as Plugin">
    Install ccprompts directly in Claude Code for instant access to all commands:
    
    ```bash
    /plugin marketplace add ursisterbtw/ccprompts
    /plugin install ccprompts@ursisterbtw
    ```
  </Step>
  
  <Step title="Analyze Your Project">
    Let ccprompts analyze your codebase and recommend relevant commands:
    
    ```bash
    /analyze-project
    ```
  </Step>
  
  <Step title="Explore Commands">
    Browse the [Command Reference](/reference/analyze-project) to discover 70+ available commands organized by development phase
  </Step>
</Steps>

## Why ccprompts?

<AccordionGroup>
  <Accordion title="Comprehensive Command Library">
    70+ production-ready commands covering every phase of software development from initial analysis to enterprise-scale operations. Commands are organized by workflow phase for easy discovery.
  </Accordion>
  
  <Accordion title="Specialized AI Agents">
    10 domain-specific agents with deep expertise in popular languages and frameworks. Each agent follows a 7-step methodology with color-coded categorization and consistent structure.
  </Accordion>
  
  <Accordion title="Enterprise-Grade Safety">
    Dagger-based container isolation ensures dangerous commands execute safely with automatic cleanup, rollback validation, and comprehensive audit trails.
  </Accordion>
  
  <Accordion title="Multi-Dimensional Validation">
    Every command is validated across four dimensions: structural integrity, security compliance, quality metrics, and performance benchmarks with numeric scoring.
  </Accordion>
  
  <Accordion title="Team-Wide Consistency">
    Install as a plugin for automatic team-wide distribution. Shared plugin configuration ensures every team member has access to the same workflows.
  </Accordion>
</AccordionGroup>
