# Best-Practices

Provides comprehensive best practices analysis, recommendations, and enforcement for development workflows. Delivers authoritative, educational, and actionable guidance with automated detection and remediation capabilities.

## Usage

```bash
/best-practices [domain] [technology] [scope]
```

## Parameters

<ParamField path="domain" type="string">
  security, performance, testing, architecture, documentation, git, accessibility, maintainability
</ParamField>

<ParamField path="technology" type="string">
  javascript, typescript, python, react, vue, angular, node, django, flask, fastapi, go, rust, java, kotlin, swift, csharp, php, ruby
</ParamField>

<ParamField path="scope" type="string">
  project, team, organization, industry
</ParamField>
