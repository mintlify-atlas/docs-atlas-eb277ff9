# Knowledge-Base

Comprehensive knowledge base management system that creates, maintains, and organizes organizational knowledge across multiple formats and platforms. Handles automated content extraction, intelligent organization, and team collaboration workflows.

## Usage

```bash
/knowledge-base [action] [scope] [format]
```

## Parameters

<ParamField path="**action**" type="string">
  create, update, search, organize, archive, analyze, extract, sync, backup, audit
</ParamField>

<ParamField path="**scope**" type="string">
  project, team, organization, department, external
</ParamField>

<ParamField path="**format**" type="string">
  wiki, markdown, confluence, notion, sphinx, gitbook, mdbook, docusaurus
</ParamField>
