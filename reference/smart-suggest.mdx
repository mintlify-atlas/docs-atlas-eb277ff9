# Smart-Suggest

Uses intelligent analysis to provide personalized prompt recommendations: 1. Analyzes current project state, recent activities, and development patterns 2. Considers team productivity metrics and common bottlenecks 3. Suggests optimal prompt sequences for maximum impact 4. Provides contextual timing recommendations for suggested actions 5. Learns from past prompt usage and success patterns 6. Adapts suggestions based on project phase and priorities

## Usage

```bash
/smart-suggest
```

## Examples

<CodeGroup>

```bash Basic Usage
# Get smart recommendations for current project
/smart-suggest
# Example output:
# IMMEDIATE ACTIONS (Today):
# - /quick-fix security - Critical vulnerability detected
```

</CodeGroup>
