## Use Cases

### Debug Complex Issue

```bash
/ai-debug "NullPointerException in user service" production
```

AI analyzes stack traces, code patterns, and historical data to identify root cause and suggest fixes.

### Learn New Framework

```bash
/ai-mentor react hooks intermediate
```

Personalized mentorship for learning React hooks with hands-on examples and feedback.

### Pair Programming Session

```bash
/ai-pair-program "implement payment gateway" realtime
```

Real-time AI assistance during payment gateway implementation with suggestions and validation.

### Understand Legacy Code

```bash
/code-explain "src/legacy/processor.js" comprehensive
```

Deep explanation of legacy code including intent, dependencies, and refactoring opportunities.

### Generate Feature Code

```bash
/code-generate "REST API for user management" comprehensive
```

Generate complete REST API with CRUD operations, validation, and tests.

### Detect Issues

```bash
/pattern-detect full-codebase all-patterns
```

Comprehensive pattern analysis across codebase for improvements and issues.

### Predictive Analysis

```bash
/predictive-dev analyze next-sprint
```

Predict potential issues and opportunities for upcoming sprint.

### Semantic Refactoring

```bash
/refactor-semantic "user-service" modernize-patterns
```

Refactor user service using semantic understanding to preserve intent while modernizing.

### Code Analysis

```bash
/semantic-understand "payment-module" deep-analysis
```

Deep semantic understanding of payment module architecture and behavior.

### Intelligent Test Generation

```bash
/test-intelligent "auth-service" comprehensive
```

AI-generated comprehensive test suite with edge cases and scenarios.
