## Use Cases

### Learn Framework Best Practices

```bash
/best-practices react production
```

Get production-ready React best practices including performance, security, and architecture.

### Build Knowledge Base

```bash
/knowledge-base create comprehensive
```

Create searchable knowledge base from existing documentation and code.

### Fix Common Issue

```bash
/quick-fix "npm install fails"
```

Rapidly diagnose and fix common npm installation issues.

### Generate Release Notes

```bash
/release-notes v2.1.0 detailed
```

Automatically generate detailed release notes from Git history.

### Get Smart Suggestions

```bash
/smart-suggest current-file
```

Receive context-aware suggestions for improving the current file.

### Plan Sprint

```bash
/sprint-planning next 2-weeks
```

Plan the next sprint with estimation, capacity, and dependency analysis.

### Validate Environment

```bash
/validate-environment production
```

Comprehensive validation before production deployment.

## Example Workflows

### New Developer Onboarding

1. **Best Practices**: `/best-practices python project-specific`
2. **Knowledge Base**: `/knowledge-base search "getting started"`
3. **Environment**: `/validate-environment development`
4. **Quick Fixes**: Address any environment issues
5. **First Task**: Begin with guided development

### Pre-Release Checklist

1. **Validate**: `/validate-environment production`
2. **Release Notes**: `/release-notes v1.5.0 comprehensive`
3. **Best Practices**: Review production checklist
4. **Quick Fixes**: Resolve any pre-flight issues
5. **Deploy**: Proceed with deployment

### Sprint Kickoff

1. **Planning**: `/sprint-planning next 2-weeks`
2. **Estimation**: Review and refine estimates
3. **Dependencies**: Identify blockers
4. **Capacity**: Balance team workload
5. **Commit**: Finalize sprint commitments
