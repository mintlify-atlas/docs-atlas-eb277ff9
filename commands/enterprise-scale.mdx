## Use Cases

### Enterprise Analytics Dashboard

```bash
/analytics-advanced organization quarterly
```

Generate comprehensive analytics dashboard for entire organization with quarterly trends.

### Multi-Framework Compliance

```bash
/compliance-enterprise "SOC2,HIPAA,GDPR" continuous
```

Continuous compliance monitoring across multiple regulatory frameworks.

### Organization-Wide Governance

```bash
/governance enforce organization-policies
```

Enforce governance policies across all teams and repositories.

### Enterprise Knowledge Base

```bash
/knowledge-org build comprehensive
```

Build organization-wide searchable knowledge base from all documentation.

### Multi-Repo Operations

```bash
/multi-repo update-dependency "react@18" all-repos
```

Update React dependency across all repositories in organization.

### Resource Optimization

```bash
/resource-manage optimize cloud-costs
```

Analyze and optimize cloud resource allocation and costs.

### Scale Performance

```bash
/scale-optimize builds enterprise
```

Optimize build and test times for enterprise-scale codebases.

### Team Coordination

```bash
/team-coordinate dependencies quarterly-planning
```

Coordinate cross-team dependencies for quarterly planning.

## Example Workflows

### Quarterly Business Review

1. **Analytics**: `/analytics-advanced organization quarterly`
2. **Compliance**: `/compliance-enterprise all-frameworks audit-ready`
3. **Resource**: `/resource-manage forecast next-quarter`
4. **Knowledge**: `/knowledge-org metrics organization`
5. **Teams**: `/team-coordinate review quarterly`
6. **Report**: Generate executive summary

### Organization-Wide Migration

1. **Plan**: `/multi-repo analyze migration-impact`
2. **Coordinate**: `/team-coordinate plan migration`
3. **Governance**: `/governance approve migration-plan`
4. **Execute**: `/multi-repo migrate all-repos`
5. **Monitor**: `/analytics-advanced migration-progress`
6. **Validate**: `/compliance-enterprise verify post-migration`

### New Team Onboarding

1. **Knowledge**: `/knowledge-org onboarding new-team`
2. **Governance**: `/governance assign team-policies`
3. **Resources**: `/resource-manage allocate team-resources`
4. **Setup**: Configure team repositories and tools
5. **Coordinate**: `/team-coordinate integrate new-team`
6. **Monitor**: `/analytics-advanced team-ramp-up`
