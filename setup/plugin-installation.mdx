The recommended way to install ccprompts is as a Claude Code plugin. This gives you instant access to all 70+ commands and 10 specialized agents in any project.

## Quick Installation

<Steps>
  <Step title="Add the ccprompts marketplace">
    Open Claude Code and run:
    
    ```bash
    /plugin marketplace add ursisterbtw/ccprompts
    ```
    
    This adds the ccprompts GitHub repository as a plugin marketplace source.
  </Step>

  <Step title="Install the plugin">
    ```bash
    /plugin install ccprompts@ursisterbtw
    ```
    
    This downloads and installs the plugin from the marketplace.
  </Step>

  <Step title="Restart Claude Code">
    Close and reopen Claude Code to load the plugin.
    
    <Note>
    Plugin commands are only available after restarting Claude Code.
    </Note>
  </Step>

  <Step title="Verify installation">
    Check that the plugin is working:
    
    ```bash
    /help
    ```
    
    You should see ccprompts commands in the list.
  </Step>
</Steps>

## Plugin Permissions

ccprompts automatically configures these permissions:

<CodeGroup>
  ```json Allowed Permissions
  {
    "permissions": {
      "allow": [
        "Bash(git:*)",
        "Bash(npm:*)",
        "Bash(pip:*)",
        "Bash(cargo:*)",
        "Read",
        "Write",
        "Edit",
        "Glob",
        "Grep"
      ]
    }
  }
  ```

  ```json Denied Permissions
  {
    "permissions": {
      "deny": [
        "Bash(rm -rf:*)",
        "Bash(sudo:*)"
      ]
    }
  }
  ```
</CodeGroup>

<Warning>
The plugin denies dangerous operations like `rm -rf` and `sudo` commands for safety.
</Warning>

## Environment Variables

The plugin sets these optional environment variables:

```json
{
  "env": {
    "CLAUDE_CODE_PROJECT_TYPE": "command-ecosystem",
    "CLAUDE_CODE_ENABLE_MCP": "1"
  }
}
```

- `CLAUDE_CODE_PROJECT_TYPE` - Identifies the project type for context
- `CLAUDE_CODE_ENABLE_MCP` - Enables Model Context Protocol integration

## Updating the Plugin

To update ccprompts to the latest version:

<Steps>
  <Step title="Uninstall current version">
    ```bash
    /plugin uninstall ccprompts@ursisterbtw
    ```
  </Step>

  <Step title="Reinstall latest version">
    ```bash
    /plugin install ccprompts@ursisterbtw
    ```
  </Step>

  <Step title="Restart Claude Code">
    Close and reopen Claude Code to load the updated plugin.
  </Step>
</Steps>

## Uninstalling the Plugin

To remove ccprompts:

<Steps>
  <Step title="Uninstall the plugin">
    ```bash
    /plugin uninstall ccprompts@ursisterbtw
    ```
  </Step>

  <Step title="Remove marketplace (optional)">
    ```bash
    /plugin marketplace remove ursisterbtw/ccprompts
    ```
  </Step>

  <Step title="Restart Claude Code">
    Close and reopen Claude Code to complete removal.
  </Step>
</Steps>

## Troubleshooting

### Commands Not Available After Installation

<Steps>
  <Step title="Verify installation">
    ```bash
    /plugin
    ```
    
    Check if ccprompts appears in the installed plugins list.
  </Step>

  <Step title="Restart Claude Code">
    **Required:** Plugins only load after restarting Claude Code.
  </Step>

  <Step title="Check marketplace">
    ```bash
    /plugin marketplace list
    ```
    
    Ensure `ursisterbtw/ccprompts` is in the marketplace list.
  </Step>
</Steps>

### Permission Errors

If you encounter permission errors:

1. Trust the repository folder where you're using ccprompts
2. Review the command's required permissions
3. Check your `.claude/settings.json` for conflicts

### Plugin Not Found

If installation fails with "plugin not found":

1. Verify the marketplace was added correctly
2. Check your internet connection
3. Try removing and re-adding the marketplace
4. Ensure you're using the correct plugin identifier: `ccprompts@ursisterbtw`
