## What is ccprompts?

ccprompts is a comprehensive Claude Code plugin that transforms your development workflow with **70+ production-ready slash commands** across 12 development phases, **10 specialized AI agents**, and a **Dagger-based safety system**. Whether you're analyzing a new codebase, deploying to production, or implementing enterprise governance, ccprompts provides the tools and expertise you need.

Install as a plugin for instant access to enterprise-grade workflows, or use the repository directly for customization.

<Note>
**Developer Notice**: This repository is under active development. Commands may contain bugs, breaking changes can occur between versions, and the structure may evolve. Use with caution in production environments.
</Note>

### Dagger-Based Safety System

ccprompts includes a comprehensive safety system for executing potentially dangerous commands:

<Steps>
  <Step title="Pattern Detection">
    Automatically identifies dangerous commands using regex patterns for destructive operations
  </Step>
  
  <Step title="Container Isolation">
    Executes risky operations in isolated Ubuntu containers with resource limits and read-only project mounting
  </Step>
  
  <Step title="Automatic Cleanup">
    Container resources are automatically released after execution with comprehensive cleanup
  </Step>
  
  <Step title="Rollback Validation">
    Verifies safety before allowing changes to persist with automatic rollback capabilities
  </Step>
</Steps>

<Warning>
To enable the safety system, install Dagger. While not explicitly required, it's highly recommended for containerized command execution.
</Warning>

### Multi-Dimensional Validation

Every command in ccprompts is validated across four dimensions:

1. **Structural Integrity** - Command format, metadata, and documentation completeness
2. **Security Compliance** - Dangerous patterns, permission requirements, input validation
3. **Quality Metrics** - Code organization, error handling, best practices
4. **Performance Benchmarks** - Resource usage, execution time, optimization opportunities

Each dimension receives a numeric score, ensuring consistent quality across the entire command library.
